JAVA EE TEKNOLOJİLERİ EĞİTİMİ

Bu eğitimde kurumsal uygulama geliştirme projelerinde yaygın biçimde kullanılan Java EE teknolojileri anlatılmaktadır. Eğitim programımız ile katılımcıların Java EE teknolojilerinin temellerini oluşturan JDBC, Servlets, JNDI, RMI, JMS gibi konu ve kavramlar, EJB teknolojileri ve Java EE projelerinde yaygın biçimde kullanılan tasarım örüntüleri hakkında bilgi sahibi olmaları hedeflenmektedir. Eğitim sonunda hangi framework veya kütüphaneyi tercih ederseniz edin Java EE teknolojileri ile kurumsal uygulama geliştirme faaliyetlerinizin çok daha verimli bir hal alacağına eminiz.

  • Java JDBC API

  • REST Tabanlı Web Servisleri (JAX RS)

Opsiyonel Konular

Eğitim süresince ana konuların yanı sıra katılımcıların ihtiyaçlarına, bilgi ve tecrübe düzeylerine, eğitim süresine bakılarak opsiyonel ve ileri düzey konu başlıklarından uygun görülen konulara da değinilmektedir.

Eğitim için önerilen süre 4 gündür. Eğitim boyunca konu anlatımları lab çalışmaları ile desteklenmektedir. Lab çalışmaları katılımcılarla birlikte gerçekleştirilmektedir.
