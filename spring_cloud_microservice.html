SPRING CLOUD VE MICROSERVICE EĞİTİMİ

Bu eğitimde genel olarak microservice mimarisinde uygulama geliştirme ile ilgili teknoloji bağımsız konuların yanı sıra Spring Cloud ekosistemindeki teknolojileri kullanarak microservice mimarisine sahip uygulamalar geliştirmeyi öğreniyoruz. Eğitim sonunda Spring Cloud ekosisteminde microservice mimarisinde uygulamalar geliştirme konusunda çok daha sağlıklı bir bilgi düzeyine gelmiş olacağınıza eminiz.

Eğitim İçeriği

Microservice Konuları
  • Microservice Mimarisine Giriş
  • İletişim Modelleri
  • Veri Erişimi ve Transaction Yönetimi
  • Service Discovery Kabiliyeti
  • API Gateway
  • Circuit Breaker ve Hata Yönetimi
  • Loglama ve İz Sürme İşlemleri
  • Versiyon ve Konfigürasyon Yönetimi
  • Güvenlik
  • Kurulum İşlemleri
  • Kullanıcı Arayüzleri
  • Container Kavramı ve Docker
Spring Cloud Konuları
  • Spring Cloud'a Giriş
  • Spring Cloud Eureka Service Discovery
  • Spring Cloud Bus ve AMQP
  • Spring Cloud Konfigürasyon Yönetimi
  • Spring Cloud Hystrix Circuit Breaker
  • Spring Cloud Ribbon Load Balancer
  • Spring Cloud Zuul API Gateway
  • Spring Cloud Sleuth/Zipkin ile Log ve Trace Yönetimi
  • Spring Cloud Security

Eğitimin Süresi

Eğitim için önerilen süre 5 gündür. Eğitim boyunca konu anlatımları lab çalışmaları ile desteklenmektedir. Lab çalışmaları katılımcılarla birlikte gerçekleştirilmektedir.

Gereksinimler


Bu eğitime katılacak kişilerin Spring Application Framework, Spring Security Framework gibi teknolojiler hakkında bilgi ve deneyim sahibi olmaları, Java Programlama Dili'ne hakim olmaları, Java ile tercih ettikleri herhangi bir IDE içerisinde rahatlıkla kod yazabilir durumda olmaları eğitimden maksimum ölçüde faydalanabilmeleri için önemlidir.
