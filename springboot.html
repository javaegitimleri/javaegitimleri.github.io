SPRING BOOT EĞİTİMİ

Bu eğitimde Spring ekosistemindeki teknolojiler ve başta JPA/Hibernate olmak üzere 3rd party diğer pek çok framework ve kütüphane ile daha kolay ve verimli çalışmayı hedefleyen Spring Boot platformunu kapsamlı biçimde öğreniyoruz. Eğitim boyunca konu anlatımları, örnek proje üzerinden yapılan egzersizlerle desteklenmektedir. Eğitim sonunda Spring ekosisteminde çalışmanın Spring Boot ile çok daha kolay ve hızlı olduğunu öğreneceğinize ve Spring Boot'u kurumsal uygulama geliştirme faaliyetlerinizde vazgeçilmez bir araç olarak görmeye başlayacağınıza eminiz.

Eğitim İçeriği

  • Spring Boot'a Giriş ve Temelleri
  • Spring Boot ve Web MVC
  • Spring Boot ve REST
  • Spring Boot ve Testler
  • Spring Boot ve Veritabanı İşlemleri
  • Spring Boot ve Data Projesi
  • Spring Boot ve Güvenlik
Opsiyonel Konular
  • Spring Boot ve Cache
  • Spring Boot ve Validasyon
  • Spring Boot ve AMQP ile Mesajlaşma
  • Spring Boot ve Task Execution/Scheduling
  • Spring Boot ve E-Posta Gönderimi
  • Spring Boot ve JSF Entegrasyonu

Eğitim süresince ana konuların yanı sıra katılımcıların ihtiyaçlarına, bilgi ve tecrübe düzeylerine, eğitim süresine bakılarak opsiyonel ve ileri düzey konu başlıklarından uygun görülen konulara da değinilmektedir.

Eğitimin Süresi

Eğitim için önerilen süre 3 gündür. Eğitim boyunca konu anlatımları lab çalışmaları ile desteklenmektedir. Lab çalışmaları katılımcılarla birlikte gerçekleştirilmektedir.

Eğitimin Faydaları


Eğitim sonunda katılımcılar;

  • Spring Boot ile executable Java web uygulamaları oluşturmayı
  • Spring Boot’un otomatik konfigürasyon kabiliyetini
  • Spring Boot Starter’ları ile çalışmayı
  • Spring Boot Actuator modülünün neler sağladığını
  • Uygulamanın konfigürasyon property’lerinin yönetimini
  • Uygulamanızı farklı hedef ortamlarda herhangi bir değişiklik yapmadan çalıştırmayı
  • Loglama kabiliyetini devreye almayı ve uygulamanın davranışını trace etmeyi
  • Spring Boot ile RESTful web servisleri geliştirmeyi
  • JSP tabanlı dinamik web sayfaları geliştirmeyi
  • Hata sayfalarını özelleştirmeyi
  • Uygulamaya özel Servlet ve Filter yazmayı
  • Entegrasyon testleri yazmayı ve çalıştırmayı
  • Farklı DataSource konfigürasyonları yapmayı
  • Veritabanı bağlantı havuzu kullanmayı
  • Gömülü veritabanları ve H2 ile çalışmayı
  • JDBC API veya JPA/Hibernate ile veritabanı işlemleri geliştirmeyi
  • Web uygulamaları için sağlanan default güvenlik kabiliyetlerini devreye almayı
  • Form tabanlı ve HTTP BASIC kimliklendirme kabiliyetlerini
  • Beni hatırla kabiliyetinin devreye alınmasını
  • Kullanıcı oturum yönetimini
  • Kullanıcı şifrelerinin veritabanında kriptolu saklanmasını
  • Web sayfalarının yetkilendirilmesini
  • Metot düzeyinde yetkilendirme kabiliyetini
  • Önbellek ve validasyon kabiliyetlerini
  • E-Posta göndermeyi, ve geliştirme ortamında e-posta göndermeyi test etmeyi


ve bunların yanında kurumsal yazılım geliştirme ve Spring Boot’la ilgili diğer pek çok konuyu öğrenme fırsatını yakalamış olacaklar.

Gereksinimler


Spring Boot Eğitimi'ne katılacak kişilerin Spring Application Framework, Spring Security Framework ve JPA/Hibernate Persistence Framework gibi teknolojiler hakkında bilgi ve deneyim sahibi olmaları, Java Programlama Dili'ne hakim olmaları, Java ile tercih ettikleri herhangi bir IDE içerisinde rahatlıkla kod yazabilir durumda olmaları eğitimden maksimum ölçüde faydalanabilmeleri için önemlidir.
