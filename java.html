JAVA PROGRAMLAMA DİLİ EĞİTİMİ

Bu eğitim ile katılımcılarımıza Java programlama dilini ve nesne yönelimli programlamayı kapsamlı biçimde öğretiyoruz. Eğitim sonunda katılımcıların, projelerinde Java programlama dilini çok daha bilinçli bir şeklinde kullanacaklarına ve projelerinde karşılaştıkları problemlere daha efektif çözümler üreteceklerine eminiz.

Eğitim İçeriği

  • Java Programlama Diline Giriş
  • Kalıtım (Inheritance)
  • Sınıf Yükleme ve Nesne Yaratım İşlemleri
  • Java Primitif ve Wrapper Tipleri
  • String ve Array İşlemleri
  • İlişkisel, Mantıksal, Şarlı İfadeler ve Döngüler
  • Java Exceptions
  • Annotations
  • Enums
  • Java Collection API
  • Inner Sınıflar
  • Java Generics
  • Java Threads & Concurrency
  • Java I/O API
  • Java Garbage Collection
  • Java Date/Time API
Opsiyonel Konular
  • Maven ile Build İşlemleri ve Bağımlılık Yönetimi
  • Internationalization & Localization (I18N)
  • JavaBeans API
  • Swing/AWT ile GUI Programlama
  • Java ve XML
  • Java ve Yardımcı Sınıflar
  • Java ve Loglama İşlemleri
  • JUnit ve Birim Testler
İleri Düzey Konular
  • Java Reflection API
  • Java Threads & Concurrency
  • JNI
  • Java ve Security
  • Lambda İfadeleri
  • Stream API

Eğitim süresince ana konuların yanı sıra katılımcıların ihtiyaçlarına, bilgi ve tecrübe düzeylerine, eğitim süresine bakılarak opsiyonel ve ileri düzey konu başlıklarından uygun görülen konulara da değinilmektedir.

Eğitimin Süresi

Eğitim için önerilen süre 5 gündür. Eğitim boyunca konu anlatımları lab çalışmaları ile desteklenmektedir. Lab çalışmaları katılımcılarla birlikte gerçekleştirilmektedir.

Eğitimin Faydaları


Eğitim sonunda katılımcılar;

  • Abstraction, encapsulation, inheritance ve polymorphism gibi temel nesne yönelimli programlama kavramlarını
  • Inheritance ve composition arasındaki farkı
  • Garbage Collector’un nasıl çalıştığını
  • Collection APIsini, List, Set, Map kullanmayı, bunların özelliklerini, birbirlerine karşı üstünlüklerini
  • Hataların ele alınmasını, exception tiplerini, kayıp exception problemini
  • Reflection APIsi ile çalışmayı, dinamik proxy oluşturmayı
  • IO işlemlerini, dosyadan okuma ve dosyaya yazmayı, tampon bellek ile çalışmayı
  • Eş zamanlı işlemleri, Thread oluşturmayı, daemon thread kavramını
  • JavaBeans bileşen modelini, bean introspection kavramını


ve daha pek çok önemli konuyu öğrenecekler, bu konularla paralel olarak yapılacak laboratuvar çalışmalarında öğrendiklerini pratik yaparak pekiştirme fırsatı bulacaklardır.
