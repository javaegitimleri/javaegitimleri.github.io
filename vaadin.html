VAADIN UI FRAMEWORK EĞİTİMİ

Günümüzde kurumsal uygulamalar için web de fakto platform haline geldi. Web tabanlı uygulamaların erişimi de tarayıcılar üzerinden gerçekleştirilmektedir. Kurumsal web uygulamaları tamamen web ortamına aktarılırken, bu uygulamaların karmaşık ve zengin içerikli kullanıcı arayüz gereksinimlerini karşılamak için zaman içerisinde evrilen çeşitli çözümler ortaya çıkmıştır. Servlet ve JSP teknolojisi ile başlayan süreç, JSF, Ajax ve çok çeşitli JavaScript frameworkleri şeklinde evrilmiştir. Özellikle ajax teknolojisi ile sayfalar arasında geçiş mantığı üzerine kurulu, etkileşim olarak da temel HTML bileşenlerinin kabiliyetleri ile sınırlı web arayüzleri, masaüstü uygulamalar gibi zengin içerikli ve etkileşimli bir hale dönüşmüştür. Ajax'ın da temeli Javascript'tir. JSP, JSF gibi taglib tabanlı ve ajax'ı mimarilerinin merkezine almayan çözümlerde javascript kullanımı oldukça problemli ve bakımı zor bir hale dönüşebilmektedir. Salt javascript tabanlı çözümler ise geliştiriciler için farklı bir programlama modeline ve ortamına aşina olmayı gerektirmete, çok farklı zorluk ve problemlere neden olmaktadır. Vaadin UI Framework ile hiç JavaScript kodu yazmaya gerek duymadan Java kodu yazarak zengin içerikli ve etkileşimli web uygulamaları geliştirmek mümkündür. Swing programlama modelini web ortamına birebir taşıyan bir framework olan Vaadin ile bakımı kolay, yeniden kullanılabilir ve test edilebilir kullanıcı arayüzlerini çok hızlı biçimde geliştirebilirsiniz.
